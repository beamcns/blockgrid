# Block Grid

Block Grid is a polished browser puzzle game built with React, TypeScript, Vite, and Tailwind CSS. Place block pieces on an 8x8 grid, clear full rows and columns, build combos, and chase your best score.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

Open the local URL printed by Vite.

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Features

- 8x8 puzzle board
- Three random pieces per round
- Desktop drag and mobile touch placement
- Placement preview and invalid placement feedback
- Smooth row and column clear animations
- Combo scoring and local best score storage
- Restart flow and game-over modal
- Lightweight generated sound effects
